# statquest-ml-theory

Implementations of machine learning concepts from The StatQuest Illustrated Guide to Machine Learning and the StatQuest YouTube channel, written from scratch in Python and NumPy.

### What's in the notebook
- Standardisation of numerical features
- Statistical measures: SSR, MSE, R²
- Gradient descent for linear regression with multiple independent variables
- k-fold cross-validation

### Dataset

I used this [student performance dataset](https://www.kaggle.com/datasets/nikhil7280/student-performance-multiple-linear-regression), which is designed to work well with simple linear regression. The goal wasn't to spend time on preprocessing or feature engineering but to see how the basic algorithms work under the hood.
